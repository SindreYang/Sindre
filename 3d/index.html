    rotation_crown(near_mesh, jaw_mesh) 

    调整单冠的轴向
    Tip

    1.通过连通域分割两个邻牙;

    2.以邻牙质心为确定x轴；

    3.通过找对颌最近的点确定z轴方向;如果z轴方向上有mesh，则保持原样，否则将z轴取反向;

    4.输出调整后的牙冠

    Parameters:

    Name Type Description Default
    near_mesh Mesh

    两个邻牙组成的mesh

    		 required
    jaw_mesh Mesh

    两个邻牙的对颌

    		 required

    Returns:

    Type Description
    Mesh

    变换后的单冠mesh
